About DeepFake Detector

Project Overview

The DeepFake Detection System is an educational web application designed to identify AI-generated fake images using advanced neural network technology.

This project was developed as part of a software engineering initiative to combat the growing threat of deepfakes in digital media. Our system uses the MesoNet convolutional neural network architecture specifically designed to detect facial manipulations in images.

System Architecture

The DeepFake Detection System utilizes a modular architecture with four main components:

  1. User Interface Module: Handles user interactions, image uploads, and results display
  2. Preprocessing Module: Prepares images for analysis (resizing, normalization)
  3. MesoNet Model: Performs the deepfake detection using convolutional neural networks
  4. Results Management: Stores and retrieves detection results
